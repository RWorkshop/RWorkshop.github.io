\begin{slide}{Binomial coefficients} \begin{itemize} \item choose() can be used to determine the number of ways $k$objects can be chosen from $n$. \begin{equation} n \choose k = \frac{n!}{k!(n-k)!} \end{equation} The command takes the form \begin{verbatim}choose(n,k)\end{verbatim} \begin{verbatim} > choose(9,2) [1] 36 \end{verbatim} The factorial of a number can be computers using the factorial() command. \begin{verbatim} > factorial(4) [1] 24 \end{verbatim} \end{slide}
