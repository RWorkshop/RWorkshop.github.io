Binomial coefficients

choose() can be used to determine the number of ways $k$objects can be chosen from \begin{equation} n \choose k = \frac{n!}{k!(n-k)!} \end{equation} The command takes the form 
choose(n,k)


> choose(9,2)
[1] 36

The factorial of a number can be computers using the factorial() command. 


> factorial(4)
[1] 24
