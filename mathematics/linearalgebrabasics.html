Before we begin, let's set up some matrices to work with. 

x1 <- c(1,2,4);x2 <- c(3,8,4);x3 <- c(1,4,3)
A <- rbind(x1,x2,x3); C <- cbind(x1,x2,x3)
b <- c(1,5,4)
rownames(A) <- NULL;

Inverting a matrix

To invert a matrix we use the command solve() with no additional argument. Remember - Not all matrices are invertible. It the determinant of a matrix is zero, then no inverse exists. 
solve(C)

##          [,1]        [,2]       [,3]
## x1  0.5714286 -0.35714286  0.2857143
## x2  0.7142857 -0.07142857 -0.1428571
## x3 -1.7142857  0.57142857  0.1428571
We can use this same command to solve a system of linear equations Ax=b. We would specify the vector b as the additional argument.

Computing the determinant

To compute the determinant, the command is simply det()

Solving a System of Linear Equations

To solve a system of linear equations in the form Ax=b , where A is a square matrix, and b is a column vector of known values, we use the solve() command to determine the values of the unknown vector x. 
solve(A, b)

## [1] -2.7142857  1.5714286  0.1428571
