Shapiro-Wilk Test

The first test we will look at is the Shapiro-Wilk test. 
X <- rnorm(10,100,2)
shapiro.test(X)

## 
## 	Shapiro-Wilk normality test
## 
## data:  X
## W = 0.94763, p-value = 0.6406
As the p-value is very high, we fail to reject the null hypothesis that the data set ``Vec" is normally distributed.
Normal Probability Plots
qqnorm(X ,pch=18,col="red")  #draws a QQ plot that is used to check for normality.
qqline(X)  #adds trend line to QQplot.
