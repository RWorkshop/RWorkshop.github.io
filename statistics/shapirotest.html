Shapiro Wilk Test

\begin{itemize} \item Used to determine if a data set is normally distributed \item Null hypothesis is that it is normally distributed. \item Alt. hypothesis is that it is not normally distributed. \item Advise using with graphical complement. \item Output return test statistic (W) and p-value. \end{itemize} \begin{verbatim} > shapiro.test(A) Shapiro-Wilk normality test data: A W = 0.961, p-value = 0.8085 \end{verbatim}
