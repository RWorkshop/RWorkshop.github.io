Section 4: Dates

As.Date Function (4.1) The as.Date function allows a variety of input formats through the format=argument. The default format is a four-digit year, followed by a month, then a day, separated by either dashes or slashes. table(mons)

The Chron Package (4.2)

The chron function converts dates and times to chron objects. The dates and times are provided to the chron function as separate values, so some preprocessing may be necessary to prepare input date/times for the chron function. The chron package converts dates and times to chron objects. 

library(chron)
dtimes=c(“2002-06-09 12:45:40”)
dtparts=t(as.data.frame(strsplit(dtimes,’ ‘)))
rownames(dtparts=NULL
Chron values are stored internally as the fractional number of days from January 1 1970.
